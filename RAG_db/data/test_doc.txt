and knowledge retrieval.

- Technical Lead: Rahul Mehta
- Goal: Build a Retrieval-Augmented Generation (RAG) system

--------------------------------------------------
Section 4: System Infrastructure
--------------------------------------------------
The RAG system for Project Atlas uses:
- PDF and CSV documents as input
- Text chunking with overlap
- Vector embeddings stored in a vector database

Vector databases under evaluation:
- FAISS
- ChromaDB
